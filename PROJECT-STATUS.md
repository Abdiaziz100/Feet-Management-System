## 🚀 HOW TO RUN

### Quick Start
```bash
./run-system.sh
```

### Manual Setup
```bash
# Setup (one time)
./setup.sh

# Start system
./start.sh
```

### Individual Components
```bash
# Backend only
cd backend && python app.py

# Frontend only  
cd frontend && npm run dev
```
